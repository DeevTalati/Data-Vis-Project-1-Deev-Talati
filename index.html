How To Use This Application

In this application, there are three different types of maps: Choropleths, Histograms, and a Scatter Plot. To use the Choropleth and Histogram, simply select the desired attribute you would like to view in the dedicated drop-down menu. For the Scatter Plot, you will need to select an attribute for both the X and Y axis. Once this is done, you will see the data populate.

Additionally, you can interact with the Scatter Plot by brushing the data to zoom into any desired section. Simply left-click and drag your mouse to the data you would like to zoom in to. To delete the selection box, just left-click your mouse.
