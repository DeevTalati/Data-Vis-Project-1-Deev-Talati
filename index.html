How To Use This Application

Within this application, users have access to three distinct types of maps: Choropleths, Histograms, and Scatter Plots. To utilize the Choropleth and Histogram functionalities, simply select the desired attribute from the dedicated drop-down menu. For both of these chart types, users can hover over the data points to obtain more specific information. For the Scatter Plot, users must select attributes for both the X and Y axes. Upon completion, the data will populate accordingly.

Furthermore, interaction with the Scatter Plot is facilitated through data brushing, enabling users to zoom into specific sections. To utilize this feature, left-click and drag the mouse over the desired data area. Deleting the selection box is as simple as left-clicking the mouse again.
